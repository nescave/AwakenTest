# AwakenShooter

AwakenShooter is a first-person shooter prototype created as a **test task** for a recruitment process.

The project demonstrates core FPS gameplay systems implemented in Unreal Engine 5.

---

## Project Overview

**Engine:** Unreal Engine 5.7.1
**Platform:** Windows (64-bit)  

Implemented systems include:
- Simple player movement with parkour elements
- Enemy AI with perception
- Round-based gameplay flow
- Visual and audio elements serving as feedback for the player
- Gameplay Abilities / tasks (where applicable)

---

## How to Run

### Option 1: Packaged Build
1. Download the build from `https://drive.google.com/drive/folders/1ts94w83dTY-g8cnHzJAvN4X_R7dzqqMi?usp=drive_link`
2. Unpack the zip file.
2. Run `AwakenShooter.exe`

### Option 2: From Source
1. Clone the repository
2. Build the project from source with UE version 5.7.1 or higher
3. Open `AwakenShooter.uproject`
4. Run from the editor on `LV_ClearAndEscape`

> **Note:** The project was developed and tested on Windows.
